# 日常工作日志系统

一个简单易用的日常工作记录工具，基于浏览器运行，无需安装。

## 功能特点

✅ **完整的字段支持**
- 📅 日期 & ⏰ 时间
- 📋 工作内容
- 🔧 备件清单
- 🛠️ 工具列表
- ⚠️ 注意事项
- 📌 工作步骤
- 📷 工作图片（支持多张上传）

✅ **数据管理**
- 本地存储（localStorage），关闭浏览器数据不丢失
- 按日期倒序排列，方便查看最新记录
- 关键词搜索功能
- JSON格式数据导出

## 使用方法

1. 双击打开 `index.html` 文件
2. 浏览器会自动加载应用
3. 点击"新增工作日志"开始记录
4. 填写完信息后点击"保存"
5. 点击右上角"导出数据"可备份所有记录

## 文件说明

```
worklog/
├── index.html    # 主程序（直接双击打开）
├── SPEC.md       # 功能规格说明
└── README.md     # 使用说明
```

## 技术说明

- 前端框架：TailwindCSS (CDN)
- 数据存储：浏览器 localStorage
- 图片存储：Base64 编码
- 无需服务器，纯客户端运行

## 注意事项

- 图片以Base64格式存储，占用存储空间
- 建议定期使用"导出数据"功能备份重要记录
- 清除浏览器数据会导致日志丢失
